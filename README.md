# InvManagementSystem

To set up the program: 
1. ensure you have all included libraries by running pip install -r requirements.txt
2. run main.py, this will open a command line interface, with intuitive menu selection.
3. You can proceed to sign up, or login using the example account (Lukekello18:Password)
4. This will direct you to the main menu, select view table, and then explore the functionality.


Editing the csv data directly:
To edit the csv files, simply open them, and copy paste new data in.
